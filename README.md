## About the Game

Pexeso is a classic memory game where the main goal is to find pairs of identical cards. To make the gameplay more engaging and less repetitive, this version is designed with a **ninja-themed setting** and a simple **mini-story** that helps players immerse themselves in the game world and stay motivated.

Instead of just flipping cards, the player becomes a ninja who is training their memory and focus by uncovering matching symbols.

## How to Play

1. Start the game.
2. Cards are placed face down.
3. Click on two cards to flip them.
4. If the images match, the pair stays uncovered.
5. If they do not match, the cards flip back.
6. The game ends when all pairs are found.

## Motivation

The main idea behind this project was to take a well-known game concept and make it more interesting by:

* adding a thematic setting,
* creating a small narrative context,
* increasing player engagement through visual style.
